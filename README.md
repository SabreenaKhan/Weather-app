Real-time Weather Dashboard

A sleek, responsive weather application built using **Vanilla JavaScript**, **HTML5**, and **CSS3**. This project demonstrates my ability to fetch data from a REST API and dynamically update the DOM based on user input.

## ✨ Features
* **Global Search:** Fetch current weather data for any city in the world.
* **Detailed Metrics:** View temperature and weather descriptions.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop views using CSS Flexbox and Grid.
* **Error Handling:** Graceful alerts for invalid city names or network issues.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure for better SEO and accessibility.
* **CSS3:** Custom styling with a modern, glassmorphism UI feel.
* **JavaScript (ES6+):** Fetch API for asynchronous data handling and DOM manipulation.
  

## 📖 What I Learned
During the development of this project, I strengthened my understanding of:
1. **Asynchronous JavaScript:** Using `async/await` and `fetch()` to handle API calls.
2. **JSON Parsing:** Extracting specific data points from complex API response objects.
3. **Event Listeners:** Implementing "Search on Enter" and button click functionality.
4. **CSS Transitions:** Creating smooth hover effects and UI updates.
